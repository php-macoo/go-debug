# 玩家 最佳用时

攻略与操作

开始与暂停：进入页面后先点开始游戏（会向服务器登记本局）再计时；对局中可用暂停 / 「继续」。登录后成绩会带上本局凭证上报。

目标：从叠放的牌中点击选牌，牌会进入下方槽位；槽位里出现三张相同图案即消除。

操作：点选场上任意未被压住的牌。下方槽位最多 7 格，满了且无法三消则本局结束。

道具：撤回（收回上一张）、洗牌（重排场上）、移出（从槽位拿出一张暂存），次数有限，留到僵局再用。

技巧：优先消掉上层露出的牌，避免槽位被单张占满；留意计时，尽量连续消除。
